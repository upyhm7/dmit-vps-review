## 实测表现：延迟、路由、跑分

根据多个中文社区的测评数据，整理如下：

### 延迟表现

- **LAX Pro（CN2 GIA）**：国内三网平均延迟 160–200ms，晚高峰基本无明显波动
- **LAX EB（CMIN2）**：电信/联通约 180–220ms，移动略低，带宽吞吐量明显优于 Pro 系列
- **HKG Pro（CN2 GIA）**：电信约 30–50ms，联通 20–40ms，移动稍高，整体延迟是三地最低
- **TYO Pro（CN2 GIA）**：电信/联通约 80–120ms，移动走 AS58453 绕香港，延迟略高但仍稳定

### 路由质量

DMIT 洛杉矶 Pro 系列的三网路由分工很清晰：
- **电信**：出去和回来都走 GIA（AS4809），全程高速公路
- **联通**：去程 AS4837，回程 GIA
- **移动**：去程经香港 AS58453，回程 GIA

这种设计的好处是三网回程都有质量保证，不会出现某个运营商单独掉链子的情况。

### 服务器性能

DMIT 用的是 AMD EPYC 系列处理器（洛杉矶主要是 EPYC 9654 和 9004/9005 系列，香港和东京是 7003 系列），存储走 Ceph 集群 + 英特尔数据中心级 SSD，磁盘 I/O 通常能跑到 1GB/s 以上。

跑 UnixBench 的分数在同价位段属于中等偏上，不是最高，但绝对没有超售严重的问题。
